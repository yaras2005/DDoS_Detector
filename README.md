# Machine Learning-Based DDoS Detection and Monitoring System

## Project Description
This project aims to detect DDoS attacks using machine learning techniques by analyzing network traffic data and classifying it as normal or attack traffic.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit
- Joblib

## Project Structure
- `data/`: dataset files
- `src/`: source code
- `models/`: saved trained models
- `results/`: evaluation results and plots
- `screenshots/`: app screenshots
- `report/`: final report
- `presentation/`: final slides
